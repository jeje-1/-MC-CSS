2. item : flex-grow 

        flex-grow : container의 크기 증가에 따른 item의 크기 증감 비율을 지정하는 속성
        * flex-grow : 0 => container의 크기 증감해도 item의 크기가 변하지 않는다.
        * flex-grow: 1 이상 => container를 가득 채우도록 item의 크기가 결정

2. item : flex-shrink 

        flex-shrink : container의 크기 감소에 따른 item의 크기 증감 비율을 지정하는 속성
        * flex-grow : 0 => container의 크기 감소해도 item의 크기가 변하지 않는다.
        * flex-grow: 1 이상 => container의 크기가 item의 크기가 작아질 때 item의 크기가 container에 맞춰 함께 감소

4. item:flex-basis

        * flex-basis : item의 기본 크기를 결정하는 속성, default : auto
        * flex-basis를 사용해 item의 크기를 지정할 때는 모든 단위를 사용할 수 있다.
        * flex-basis의 속성값을 0으로 지정하면 해당 item은 (absolute flex item)이 되어서
          content의 크기를 기준으로 item의 크기를 결정한다.
