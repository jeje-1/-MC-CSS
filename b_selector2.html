        바로 뒤의 요소를 선택 : +
        뒤에 있는 모든 요소를 선택 : ~

반응 선택자

        사용자의 움직임에 따라 요소를 선택하는 선택자.
        1. 사용자가 클릭한 요소 선택           : 선택자:active
        2. 사용자가 마우스를 올린 요소 선택      : 선택자:hover

        1) 입력요소에 focus가 맞춰져 있는 경우
        2) 입력요소가 체크 상태인 경우

        4) 입력양식이 사용 가능한 경우

nth child

        순서로 원하는 요소를 선택하는 선택자
        선택자 : nth-child(3)   => 3번째 선택자로 지정되는 요소들 중 3번쨰
        선택자 : nth-child(2n)  => 선택자로 지정되는 요소들 중 2n번쨰 요소
