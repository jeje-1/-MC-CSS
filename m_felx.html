flex container

        container는 정렬할 요소(itme)을 감싸는 부모요소.
        display:felx, display : inline-flex 지정을 통해 해당 요소를 Container로 만들 수 있다.
        
        dispaly : flex : block 요소의 특징을 지니는 Container
        display: inline-flex : inline 요소의 특징을 지니는 Container

1. display:flex

2. display:inline-flex

3. flex-direction

3-1. flex-direction:row : 좌 -> 우

3-2. flex-direction:row-reverse : 우 -> 좌

3-3. flex-direction:column : 상 -> 하

3-3. flex-direction:column-reverse : 하 -> 상
