z-index

        z-index : 요소의 z축을 지정
        positon-static에서는 사용할 수 없다.
        
        html 요소가 쌓이는 맥락
        
        1. 브라우저가 먼저 렌더링하는 요소가 낮은 z축값을 가진다.
            * 부모요소가 자식요소보다 z축이 낮다.
            * 형제요소라면 코드라인상으로 먼저 작성된 요소가 나중에 작성된 요소보다 z축이 낮다.
            * positon 속성값이 지정된 요소가 position 속성이 지정되지 않은 요소보다 z축이 높다. 
              position 속성이 지정되지 않은 요소를 브라우저가 먼저 렌더링한다.
        z-index를 사용해 특정 요소를 상단에 배치하고 싶다면, z-index 값을 아주 크게 잡는게 좋다.
